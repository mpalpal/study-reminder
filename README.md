# Study Reminder

## 概要

これは「毎日の学習を記録し、適切なタイミングで復讐できる」ことを目的とした**個人利用向けの学習管理Webアプリ(PWA)**です。
エビングハウスの忘却曲線から発想を得ており、効率的な学習をサポートすることを期待しています。

実際に iPhone / iPad /Mac Book で日常的に使うことを想定して設計・開発しました。

---

## アプリ利用
**Web (PWA対応): *https://333dc45a.study-reminder.pages.dev***

Safari で URL を開き、**共有 → 「ホーム画面に追加」/「Dockに追加」することでアプリとして利用できます。

---

## 開発目的

- 学習内容を「毎日簡単に」記録したい
- 昨日・1週間前・2週間前など、**復習すべき内容を自然に確認したい**
- 通知に依存せず、**朝と夜に自分で開く学習習慣**を作りたい
- 広告や課金のない、**完全に個人データだけで完結するアプリ**にしたい

これらを満たすため、  
サーバーを使用しないクライアントサイド完結型設計を採用しました。

---

## 主な機能

### 復習ページ
- 昨日 / 1週間前 / 2週間前の学習内容を固定表示
- 任意の日付（当日〜30日前）を選択して復習可能
- 学習タイトルをタップすると学習メモを表示

### 記録ページ
- 当日の学習内容を「タイトル＋学習メモ」で記録
- 記録済み内容の編集・削除

### 履歴ページ
- カレンダー形式で学習履歴を一覧表示
- 過去日付の学習記録を追加・編集可能（記録し忘れた日の後追い入力に対応）

### バックアップ
- 学習データを JSON 形式でエクスポート / インポート
- 端末変更やバックアップ用途を想定

### その他
- オフライン対応
- 右上「？」ボタンからアプリの使い方を確認可能

## 設計上の工夫・こだわり

- **完全クライアントサイド設計**
  - 学習データは端末内にのみ保存され、外部に送信されない
- **UX を意識した状態管理**
  - 編集モードの明示、誤操作防止
- **日付管理の安定性**
  - iOS の日付ズレを考慮した Date 操作
- **PWA 設計**
  - ホーム画面追加、オフライン起動対応

## 想定される使い方

- 朝：復習ページで学習内容を確認
- 夜：記録ページでその日の学習を記録
- 定期的に：JSON 形式でバックアップを取得 (念の為)

---

## 注意事項

- 学習データは端末内（ブラウザ）に保存されます
- URL を知っている他人が学習内容を見ることはできません
- 別端末間での自動同期は行われません（手動インポートのみ）
